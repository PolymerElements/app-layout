##&lt;app-scrim&gt;

app-scrim is the backdrop under overlays.

Example:

```html
<app-scrim visible></app-toolbar>
```

### Styling

Custom property             | Description            | Default
----------------------------|------------------------|----------------------
`--app-scrim-color`         | Scrim color            | black
`--app-scrim-opacity`       | Scrim opacity          | 0.5
